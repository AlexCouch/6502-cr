# 6502-cr
A 6502 emulation written in crystal

## Instructions Implemented

* LDX
    * Immediate mode
    * Zero page mode
    * Absolute mode
* STX
    * Immediate mode
    * Zero page mode
    * Absolute mode
* LDY immediate mode
    * Immediate mode
    * Zero page mode
    * Absolute mode
* STY
    * Immediate mode
    * Zero page mode
    * Absolute mode
* LDA
    * Immediate mode
    * Zero page mode
    * Absolute mode
* STA
    * Immediate mode
    * Zero page mode
    * Absolute mode
* ADC
    * Immediate mode
    [ ] Zero page mode
    * Absolute mode
* JSR
* RTS
